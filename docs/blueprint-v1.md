# Papaipay Blueprint v1

## Project Overview

Papaipay is a Financial Assessment and Asset Management Platform designed to support members through financial assessment, document collection, asset matching, contribution tracking, and asset acquisition workflows.

The MVP focuses on operational efficiency, transparency, and structured member management.

---

## User Roles

### Public

- View website
- Learn about Henry's Asset
- Contact company
- Register as member

### Member

- Register account
- Complete profile
- Submit assessment
- Upload documents
- View status
- View assigned assets
- View contribution history

### Advisor

- View assigned members
- Follow up members
- Add notes
- Update progress
- Monitor applications

### Admin

- Full system access
- Manage members
- Manage advisors
- Manage assets
- Review assessments
- Manage contributions
- View reports

## MVP Objective

The primary objective of the MVP is to provide a working platform where:

- Members can register and submit assessments
- Members can upload documents
- Advisors can monitor members
- Admins can manage assets and applications
- Progress can be tracked transparently
