# PAPAIPAY Component Library V1

## Purpose

This document lists required UI prototype components for PAPAIPAY Portal Prototype V1. Components are documentation targets only until a separate implementation task is approved.

## Opportunity Components

### OpportunityCard

Displays a summarized opportunity with title, location, image, status, funding progress, target amount, participation call-to-action, and risk label.

### FundingProgressBar

Displays mock funding progress, target amount, funded amount, percentage, and status.

### OpportunityDetailTabs

Organizes detail content into overview, property information, campaign information, media, timeline, updates, distributions, FAQ, and risk disclosure tabs.

### PropertyInformationPanel

Shows property type, location, size, tenure, valuation, documents placeholder, and key property notes.

### CampaignInformationPanel

Shows campaign target, minimum participation, timeline, funding status, lifecycle status, and mock payment status labels.

### ParticipatePanel

Supports member participation intent using mock amount entry, acknowledgement checkbox, risk disclosure reminder, and prototype submit state.

## Project and Progress Components

### ProjectTimeline

Displays lifecycle milestones from draft through completed or archived states.

### ProgressUpdateCard

Displays project update title, date, category, summary, media thumbnail, and related opportunity or project.

## Media Components

### MediaGallery

Displays property images, progress images, captions, alt text, categories, and visibility labels.

## Distribution Components

### DistributionSummaryCard

Displays mock totals for expected distribution, distributed amount, pending amount, next distribution date, and completed distribution count.

### DistributionHistoryTable

Displays mock distribution rows with date, project, amount, status, reference, and notes.

## Communication Components

### AnnouncementCard

Displays announcement title, audience, priority, publish date, summary, related opportunity, and status.

### NotificationCategoryList

Groups notifications by opportunity updates, participation updates, payment status, project progress, distributions, announcements, account/security, and system notices.

## Profile Components

### ProfileTabs

Organizes profile, documents placeholder, security placeholder, preferences, and notification settings.

## Admin and Permission Components

### AdminOpportunityForm

Prototype form for opportunity title, location, property details, campaign details, lifecycle status, funding fields, media, FAQ, and risk disclosure links.

### VisibilityControlsPanel

Displays controls for draft/published state, role visibility, feature placement, announcement visibility, and archive status.

### RolePermissionMatrix

Displays Member, Manager, Admin, and Super Admin capabilities across view, create, edit, approve, publish, archive, and manage permissions.
