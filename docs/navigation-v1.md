# PAPAIPAY Navigation V1

## Purpose

This document defines target sidebar and navigation structures for PAPAIPAY Portal Prototype V1. It is documentation-only and must not be treated as approval to change UI or routes.

## Member Navigation

Primary sidebar:

1. Dashboard
2. Opportunities
3. My Participations
4. Active Projects
5. Completed Projects
6. Distributions
7. Announcements
8. Notifications
9. Media Gallery
10. FAQ
11. Risk Disclosure
12. Profile
13. Settings
14. Security

Suggested secondary opportunity detail tabs:

- Overview
- Property Information
- Campaign Information
- Media Gallery
- Timeline
- Progress Updates
- Distributions
- FAQ
- Risk Disclosure
